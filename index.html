TUTORIAL GITHUB

PARTE 1

1° - Para criar uma conta no GITHUB clique aqui e você será redirecionado para a pagina de cadastro do git

Ao entrar no link digite seu email no campo abaixo

Após colocar o email, digite uma senha

Crie um nome de usuário

Confirme se você quer receber atualizações pelo email e logo depois realize o CAPTCHA para provar que você é um humano

Após isso, você receberá um codigo no email, use-o para confirmar a conta

Depois de confirmar você cairá nessa tela de Personalização

Você pode escolher seus interesses

Por fim nessa parte, você pode escolher se quer uma conta normal ou de estudante
(para a conta de estudante tem que usar o email institucional e confirmar sua matricula por meio de algum documento)
